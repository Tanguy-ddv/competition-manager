# Competiton Manager

This project aims to create a software able to manage any type of competition.

# Introduction

This software should replace every type of paper used to create a local tournament. The goal is being able to manage every type of competition. Leagues, Cups, tournaments, Sportive competition, Artistic competiton, etc.

# Programming

This software is so far programmed in python, the interface is made with tkinter. It is planned to be turnned into a java software and a website.

# Features

- Creation and modification of competition
- 100% customizable competition
- Template of real competitions
- Language supports for French and English (at least)
- Color management
- Player statistics through the competitions
